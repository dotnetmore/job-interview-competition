Задача: 
Реализовать простой алгоритм продажи продуктов(мясо, апельсины, бабаны, клубника) на оптовой базе.
Опции: возможность сохранения/восстановления состояния по количеству продукции на базе.

Решение:
Для решения использовал паттерны Memento и Singleton, класс Stack<T>.
Результат работы программы в файле скриншота.
